My Projects

  • Outdoor Utah

    This web app enables you to share your favorite Utah outdoor activities. The app stores data in a mysql database and renders it onto the page via handlebars. The app also utilizes google's api to locate and place the activity on a map allowing users to locate the activities.

  • Spicy Taco Night

    A movie night app that allows users to get a movies and fancy taco recipe to eat/watch. We used ajax calls to hit apis fo the data.

  • Scraper

    This is a web scraper that scrapes lifehacker for its latest articles. Parts of the articles are stored in a MongoDB database and rendered through handlebars. Users can leave notes oon particular articles.

  • Eat the Burger

    This web app allows users to input and eat burgers. The app takes user input and displays it on one side of the app. Then users can eat the burger which moves that content to the other side of the application.

  • Hangman

    This is a Star Wars themed hangman. I used Jquery to target DOM elements and javascript to manipulate the data.

  • Trivia

    This is a random trivia game. The application tracks your correct and incorrect guesses. In between each question it shows a gif depending on whether or not you get the correct answer.
